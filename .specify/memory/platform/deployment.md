## Vercel routing

- **Root Directory:** `client/` on Vercel project settings
- `vercel.json` + `client/vercel.json` ŌĆö rewrites `/api/*` and `/socket.io/*` to Render
- Generated by `client/scripts/generateVercelConfig.cjs` at install/build (`RENDER_API_PROXY_URL` env or gitignored `.cursor/production-hosts.local.json`)
- **installCommand:** `HUSKY=0 node scripts/generateVercelConfig.cjs && npm install` ŌĆö must stay under `client/scripts/` (not `../scripts/` ŌĆö exit 127 on Vercel); `HUSKY=0` skips monorepo root prepare hook on Vercel
- Root `scripts/generateVercelConfig.js` ŌĆö local wrapper only
- SPA fallback: all other paths ŌåÆ `/index.html`
- **Mobile login:** committed `client/vercel.json` with real rewrite required ŌĆö placeholders break phone login
