# Obsidian Memory Template

Reusable starter for future projects that want canonical Obsidian-based agent memory.

## Usage

1. Copy this folder into target project as `memory/obsidian`.
2. Add generation/check scripts:
   - `memory:obsidian:sync`
   - `memory:obsidian:check`
3. Update project docs to point AI agents to `memory/obsidian/INDEX.md`.
4. Keep any legacy memory trees as compatibility stubs only.

## Recommended Files

- `INDEX.md`
- `Conventions.md`
- `RoutingAndAccess.md`
- `FeatureUnlocks.md`
- `LockedZones.md`
- `MasterReference.md`
- `RecentChanges.md`
